# Installing the Hyperspace API Client

You can access the Hyperspace database by installing Hyperspace's Client API into your local environment. This piece of code serves as a local client, enabling seamless communication with the Hyperspace services. This allows you to upload and manage your data, and access Vector, Classic and Hybrid similarity search capabilities.

Accessing the Hyperspace API requires credentials, provided by Hyperspace.

**To install the Hyperspace client –**

Run the following shell command in your code or your data terminal –

{% tabs %}
{% tab title="Python" %}
{% code lineNumbers="true" %}

```python
pip install hyperspace-py
```

{% endcode %}
{% endtab %}

{% tab title="Java" %}
Instruction are available under <https://github.com/hyper-space-io/hyperspace-java>
{% endtab %}

{% tab title="JavaScript" %}
{% code lineNumbers="true" %}

```javascript
npm install https://github.com/hyper-space-io/hyperspace-js
```

{% endcode %}
{% endtab %}
{% endtabs %}

**Where –**

* <mark style="color:purple;">hyperspace-py</mark> is the name of the Hyperspace's Client API that you downloaded.

Wait until the message **Successfully Installed** is displayed.
